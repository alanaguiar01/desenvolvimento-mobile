## Requisitos de Hardware
<br>

* Processador com tecnologia de virtualização
    * Intel: AV-T
    * AMD: AMD-V

Obs.: A tecnologia de virtualização deve estar habilitada na BIOS. Para verificar se está habilitada, verifique no Gerenciador de Tarefas.

<br>
